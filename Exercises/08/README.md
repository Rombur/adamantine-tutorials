Thermomechanics I
=================
The goal of this exercise is to run a thermoelastic simulation. In order to do this, you will need to modify the `input.info` file. You can run the exercise using `./adamantine -i input.info`. The solution of the exercise is given in `solution.info`.

Please refer to the documentation in the
[physics](https://adamantine-sim.github.io/adamantine/doc/input_file.html#physics-required), [discretization](https://adamantine-sim.github.io/adamantine/doc/input_file.html#discretization-required), and [boundary conditions](https://adamantine-sim.github.io/adamantine/doc/input_file.html#boundary-required).
